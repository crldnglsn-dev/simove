# 📱 SIMOVE – Driver Earnings Tracker

SIMOVE is a mobile application designed for MC taxi drivers, delivery riders, and other ride-based earners to track and manage their daily income, expenses, and overall net earnings.

---

## ✨ Features

- ➕ Input fare and customer payment per ride  
- 💰 Automatic calculation of:
  - Net earnings  
  - Platform deductions  
  - Tips  
  - Expenses  
- 🚗 Track number of rides per day  
- ⏱️ Shift timer with 8-hour warning to prevent overworking  
- 🎯 Set and monitor daily earnings goals  
- 📊 View summaries:
  - Daily  
  - Weekly  
  - Monthly  
- 📤 Export and share earnings data  
- 🌙 Dark mode support  
- 📉 Clean net earnings tracking after deductions and expenses  

---

## 🎯 Purpose

SIMOVE helps riders better understand their actual income by accounting for platform fees, expenses, tips, and deductions. It provides a clearer view of **true net earnings**, not just gross income.

## 🚀 How to Use

1. Download and install the APK from Releases  
2. Open the SIMOVE app  
3. Start adding your ride details  
4. Track earnings, expenses, and summaries  
5. Monitor your daily goals and shift duration
